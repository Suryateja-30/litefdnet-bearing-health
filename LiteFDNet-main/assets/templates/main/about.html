About LiteFDNet

Lightweight Bearing Fault Diagnosis System

LiteFDNet is a lightweight deep learning model designed for current sensor-based bearing fault diagnosis. The system analyzes motor current signals and converts them into Time-Domain Frequency (TDF) representations for effective feature extraction.

The network architecture is built using Residual Linear Blocks (RLBs) combined with dense connectivity to enhance information sharing between layers. This design improves classification performance while maintaining computational efficiency.

The final classifier predicts different bearing fault conditions based on extracted features, making the system suitable for intelligent fault monitoring and real-time industrial applications.
