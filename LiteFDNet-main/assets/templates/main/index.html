ABSTRACT

LiteFDNet: Lightweight Current Sensor-Based Fault Diagnosis

Bearings are critical components in electric motors used across industrial environments. Undetected bearing faults can lead to severe motor failures and costly system downtime.

Traditional deep learning approaches such as CNN and RNN models rely on high-dimensional raw signal data, resulting in increased computational complexity and slower inference times.

LiteFDNet introduces a lightweight approach by transforming raw current signals into optimized Time Domain Features (TDF), significantly reducing both training and inference time while maintaining high diagnostic accuracy.

  • Reduced model complexity
  • Faster inference for real-time systems
  • Efficient feature engineering using TDF
  • Industrial deployment ready
